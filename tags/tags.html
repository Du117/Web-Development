The following paragraph is a demonstartion of the cite, dfn, address, ins , del and stags.

The cite tag is used in HTMl and in webpages when you are referencing a piece of work such as a book, film, or a reasearch paper. The cite tag is used to indicate where the citationis from. For instance, the details and information about the cite tag used in the above senentence is taken from HTML & CSS design and build websites by Jon Duckett.
The dfn tag is used to indicate the defining instance of a new term. The first time you explain some new terminology (perhaps an academic concept or some jargon) in a document, it is known as a defining instance of it.
The address tag is used for a specific purpose, to contain contact details for the author of the page. It can contain physical address, a phone number and an email id for instance. Browsers often dispaly the content of the address tag in italics. There is an additional resource in HTML called the hCard microformat for adding physical address information to the webpage.
The ins, del, and s are used to make emphasis on the content of the webpage. The ins tag can be used to show content that has been inserted into a document., while the del tag can show text that has been deleted from it. The content of an ins tag is usually underlined, while the content of a del tag usually has a line through it. For example, It was the worst best idea ever, in a million years.
The s tag indicates something that is no longer accurate or relevant (but that should not be deleted). Visually the content of an s tag will usually be displayed with a line through the center. For example, Laptop for Sale: Was $3990 Now $1234.
