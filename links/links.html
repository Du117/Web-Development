Demonstration of the anchor tags in HTML for linking webpages and parts of the same webpage.



The following webpage demonstartes the use of the anchor link in HTML. The anchor tag is denoted by the "a". The anchor tag is used to link a webpage to different webpages and/or different parts within the same webpage. The a tag is a hyperlink that links webpages and parts of the webpage to the hyperlink. The most important attribute of the element is the href attribute, which indicates the link's destination.
By default links appears as one of the following in browsers,

  • An unvisited link is underlined and blue.
  • A visited link is underlined and purple.
  • An active link is underlined and red.


The following part of the webpage will help demonstrate the different tags and their various uses. Tags can be used in the following cases,

  • Use of a hyperlink to link to a different webpage.
  • Use of a hyperlink to link to a particular website, more clearly to a particular pre-determined URL.
  • Use of a hyperlink to link to different parts of the same webpage.
